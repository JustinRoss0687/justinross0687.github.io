Sudoku Solver AI

×
This project uses the implementation of the Backtracking Algorithm for solving Sudoku puzzles. This process includes the Constraint Satisfaction Problem which is a class of problems which may be represented in terms of variables (a, b, …), domains (a in [1, 2, 3], …), and constraints (a < b, …). For each backtracking step, empty cells could be assigned numbers sequentially or, by using the minimum-value-remaining heuristic (mvrh) or, by using the degree heuristic. Sequentially meaning to go through the list inorder, minimum-value-remaining heuristic meaning to use the empty cell with the least domain length, and the degree heuristic similar to the mvrh but chooses between multiple cells that have the same domain length and chooses the best option.
